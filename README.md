# introduction-to-webgl
Code samples of my internal talk at Contentful about webgl.

## Running the project
clone the project
`git clone git@github.com:Khaledgarbaya/introduction-to-webgl.git`
build the library
```bash
$ cd introduction-to-webgl/tenx2d
$ yarn
$ npm run build
```
